# Company Portfolio (Fullstack)

A fullstack web application for a company portfolio built with modern technologies.

## 🚀 Tech Stack

* Frontend: Angular + Tailwind CSS
* Backend: Express.js (Node.js)
* Database: MongoDB

## 📌 Features

* Public Website:

  * Home
  * About
  * Services
  * Projects
  * Portfolio
  * Contact

* Admin Dashboard:

  * Manage Services
  * Manage Projects
  * Manage Messages
  * Manage members
